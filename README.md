# md-export

#### version 1.0

非常轻量的md文件导出组件，只有43行代码，没有任何依赖，有用请star，谢谢

> 如何使用
```javascript
  new Md({
    mdData: `your md file content`  //require
    fileName: `generate file name`  //optional
  }).export();
```
